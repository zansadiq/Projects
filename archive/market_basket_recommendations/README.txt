 This file contains Assignment #3.
 
 The purpose of this assignment is to create a recommendation system for a company's customers based upon past purchasing history. 
 
 The data is broken into a training set of 1,000,000 records and a test set of 100. 
 
 The output is a text file with 100 lines and no headers. 
 
 Each line of the output csv contains a product recommendation based upon the corresponding purchase history in the test set. 
 
 The product recommendations are based upon the frequencies of unique combinations found in the training set.
